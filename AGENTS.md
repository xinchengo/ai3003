# AGENTS.md

This repository contains homeworks / lab experiments for the AI3003 深度学习基础课程，structure:

- `ex/`, `hw/` homework
- `lab`: 小实验

## 环境

- 所有实验需要使用电脑上的 `ai25` conda 环境，不要使用其它的环境
- LaTeX 编译要输出到 tex 所在目录的 `.build/` 子文件夹中，编译过后，不必清理文件

## 小实验

I may ask you to help out on 小实验，注意：

- 除非特别声明，仅完成作业要求中需要完成的内容；
- 代码主体部分在 `.ipynb` 中完成，每个单元格应有特定的功能；要写得简洁，像人正常写出来的；
- 报告部分：
  - 语言风格要简洁；
  - 但是对于**实验原理**必须要做足必须的分析，如果实现的某部分和报告的基本要求不一样，需要分析清原因；
  - 需要参考之前的风格（语言风格和排版风格）；
  - 确认实验要求中的**所有内容**在报告中均有体现（重要）
- 训练正常需要跑很长，如果跑不完的话，**不要减少实验参数**，应该在报告和 ipynb 中留空，等我手动运行补全；
