# USTC AI3003 Fundamentals of Deep Learning

This repository contains my implementation of the assignments for the course "Fundamentals of Deep Learning" (AI3003) at University of Science and Technology of China (USTC).

The contents of the repository is written **in Chinese**.

---

这是我在中国科学技术大学（USTC）开设的课程《深度学习基础》（AI3003）的作业实现；我修读的时期是 2026 年春季学期。

`hw` 文件夹存储的是小作业和期末作业的实现，`final.tex` 是期末作业的报告。

`lab` 文件夹存储的是若干次小实验和大实验。
